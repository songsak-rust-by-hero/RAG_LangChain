# RAG LangChain Project (โปรเจกต์ทดลอง)

## ภาพรวม

โปรเจกต์นี้เป็นระบบทดลองแบบ Retrieval-Augmented Generation (RAG) ที่ใช้แนวคิดการค้นหาข้อมูลจากเอกสาร แล้วนำข้อมูลที่เกี่ยวข้องมาให้ LLM ใช้ตอบคำถาม

เหมาะสำหรับการเรียนรู้การสร้างระบบ AI ที่เชื่อมระหว่าง “การค้นหา (Search)” และ “การสร้างคำตอบ (Generation)”

---

## ฟีเจอร์หลัก

* โหลดข้อมูลจากเว็บ
* แบ่งเอกสารเป็น chunk
* สร้าง embedding ของข้อความ
* เก็บข้อมูลใน FAISS vector database
* ค้นหาข้อมูลที่เกี่ยวข้องด้วย semantic search
* ใช้ LLM ตอบคำถามจาก context
* จำกัดการตอบให้ยึดตามข้อมูล (ลดการมั่ว)

---

## โครงสร้างการทำงาน

1. โหลดข้อมูล (Loader)
2. แบ่งข้อความ (Splitter)
3. แปลงเป็นเวกเตอร์ (Embeddings)
4. เก็บใน Vector Store (FAISS)
5. ค้นหาข้อมูลที่เกี่ยวข้อง
6. ส่ง context + คำถามให้ LLM
7. สร้างคำตอบ
